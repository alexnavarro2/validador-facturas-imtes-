El SAT bloquea llamadas directas desde navegadores (CORS). Para habilitar la validación:
  1. Instala Python 3 si no lo tienes (python.org)
  2. Ejecuta en terminal:
pip install flask requests flask-corsCopiar
3. Descarga y ejecuta el proxy:
python sat_proxy.pyCopiar
El proxy corre solo en tu equipo (localhost:5050). No expone datos externos.
Nota: Se ignoraron archivos de metadatos de macOS (._*.xml) automáticamente.
